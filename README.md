# dashboard-demo-be

## Setup

1. Install dependencies:

```bash
npm i
```

2. Create a .env file (see .env.example) with configuration

3. Initialize the database:
```bash
npm run db:init
```

### Development

Run the dev server with auto-reload:
```bash
npm run dev
```

### Production

Build TypeScript and start the server:

```bash
npm run build
npm run start
```
